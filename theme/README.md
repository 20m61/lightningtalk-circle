# 第1回 なんでもライトニングトーク - テーマ

このテーマは「第1回 なんでもライトニングトーク」イベント用のWebサイトテーマです。

## 🎯 イベント概要

- **イベント名**: 第1回 なんでもライトニングトーク
- **キャッチコピー**: 5分間で世界を変える！あなたの「なんでも」を聞かせて！
- **開催日時**: 2025年6月25日(水) 19:00〜22:00
- **会場**: 新宿会場（西新宿8-14-19 小林第二ビル8階）
- **定員**: 会場50名 / オンライン無制限
- **形式**: 5分間のライトニングトーク（最大20組）

## 📁 ファイル構成

```
lightning-talk-theme/
├── index.html              # メインページ
├── registration-form.html  # 参加・発表申込みフォーム
├── style.css              # スタイルシート
└── README.md              # このファイル
```

## 🎨 デザインコンセプト

### カラーパレット

- **プライマリ**: #FFD700 (ライトニングイエロー/ゴールド)
- **セカンダリ**: #1E3A8A (ディープブルー)
- **アクセント**: #F59E0B (アンバー)
- **テキスト**: #1F2937 (ダークグレー)
- **背景**: #F9FAFB (ライトグレー)

### 特徴

- ⚡ ライトニング（稲妻）モチーフを随所に配置
- レスポンシブデザイン対応
- アニメーション効果でダイナミックな印象
- 日本語フォントに最適化（Noto Sans JP、M PLUS Rounded 1c）

## 🚀 主な機能

1. **イベント情報表示**
   - 開催概要
   - タイムスケジュール
   - 会場情報とアクセス

2. **トピック紹介**
   - 12カテゴリーのトピック例を視覚的に表示
   - ホバーエフェクト付きカード

3. **申込みフォーム**
   - 参加者/発表者の選択式フォーム
   - 発表希望者向けの追加入力フィールド
   - バリデーション機能

4. **アニメーション効果**
   - スクロール時のフェードイン
   - ライトニングフラッシュ効果
   - ホバーエフェクト

## 💻 技術仕様

- **HTML5** / **CSS3**
- **レスポンシブ対応**（モバイルファースト）
- **Google Fonts**（Webフォント）
- **Vanilla JavaScript**（軽量実装）

## 📱 対応デバイス

- デスクトップ（1200px以上）
- タブレット（768px〜1199px）
- スマートフォン（767px以下）

## 🔧 カスタマイズ

### カラー変更

`style.css`の`:root`セクションでCSS変数を編集：

```css
:root {
  --lt-primary: #ffd700; /* メインカラー */
  --lt-secondary: #1e3a8a; /* サブカラー */
  --lt-accent: #f59e0b; /* アクセントカラー */
}
```

### コンテンツ更新

各HTMLファイル内のテキストを直接編集してください。
