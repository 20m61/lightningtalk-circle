## 🚀 はじめに

### インストール

```bash
npm install @lightningtalk/components
```

### 基本的な使い方

```tsx
import { Button, Header, EventCard } from '@lightningtalk/components';

function App() {
  return (
    <>
      <Header
        navItems={[
          { label: 'ホーム', href: '/' },
          { label: 'イベント', href: '/events' }
        ]}
      />

      <Button variant="primary" size="large">
        イベントに参加する
      </Button>

      <EventCard
        event={{
          title: '第10回ライトニングトーク大会',
          date: '2024-03-15',
          participants: 42
        }}
      />
    </>
  );
}
```

## 📚 コンポーネントカテゴリ

### Atoms（基本要素）

- **Button**: ボタンコンポーネント
- **Input**: 入力フィールド
- **Badge**: バッジ表示
- **Icon**: アイコン

### Molecules（複合要素）

- **Card**: カードコンテナ
- **Modal**: モーダルダイアログ
- **Toast**: 通知メッセージ
- **Dropdown**: ドロップダウンメニュー

### Organisms（機能単位）

- **Header**: ヘッダーナビゲーション
- **EventCard**: イベント情報カード
- **ParticipantList**: 参加者リスト
- **SpeakerDashboard**: スピーカーダッシュボード

### Templates（ページテンプレート）

- **EventPage**: イベントページレイアウト
- **DashboardLayout**: ダッシュボードレイアウト
- **LandingPage**: ランディングページ

## 🌟 ベストプラクティス

1. **レスポンシブファースト**: 全てのコンポーネントはモバイルファーストで設計
2. **アクセシビリティ**: 適切なARIA属性とキーボードナビゲーション
3. **パフォーマンス**: 遅延読み込みとコード分割を活用
4. **一貫性**: デザイントークンを使用して統一感のあるUI
5. **カスタマイズ性**: CSS変数とpropsで柔軟にカスタマイズ可能

## 🛠️ 開発ツール

- **React 18+**: 最新のReact機能を活用
- **TypeScript**: 型安全な開発
- **CSS Modules**: スコープ化されたスタイリング
- **Storybook**: コンポーネントの開発とテスト
- **Vitest**: ユニットテスト
- **Playwright**: E2Eテストとビジュアルリグレッション

## 📝 貢献ガイドライン

1. フィーチャーブランチで開発
2. コンポーネントには必ずStorybookストーリーを追加
3. テストカバレッジ80%以上を維持
4. アクセシビリティチェックを実施
5. レビュー後にマージ
