## 🚀 使用方法

### 1. インストール

```bash
# WordPressテーマディレクトリにアップロード
/wp-content/themes/lightning-talk-pro/
```

### 2. 有効化

- WordPress管理画面 → 外観 → テーマ
- Lightning Talk Pro Theme を有効化

### 3. ランディングページ設定

- 設定 → 表示設定 → ホームページの表示
- 「固定ページ」を選択
- フロントページ: Lightning Talk とは？ページを選択

### 4. コンテンツ設定

- イベントを作成（Lightning Talk イベント）
- メニューを設定（外観 → メニュー）
- サンプルデータのインポート（sample-pages.sql）
